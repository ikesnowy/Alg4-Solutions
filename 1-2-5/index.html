题目

1.2.5 以下这段代码会打印出什么？

String s = "Hello World";
s.toUpperCase();
s.substring(6, 11);
StdOut.println(s);

解答

string 类型中的 Uppercase() 以及 Substring() 都不会改变原有字符串，而是新建一个字符串。 因此输出仍然为 Hello World。

代码

using System;

namespace _1._2._5
{
    /*
     * 1.2.5
     * 
     * 以下这段代码会打印出什么？
     * String s = "Hello World";
     * s.toUpperCase();
     * s.substring(6, 11);
     * StdOut.println(s);
     * 
     */
    class Program
    {
        static void Main(string[] args)
        {
            string s = "Hello World";
            s.ToUpper();
            s.Substring(6, 5);//C# 中两个参数分别代表子串起始下标和长度
            Console.WriteLine(s);
        }
    }
}
