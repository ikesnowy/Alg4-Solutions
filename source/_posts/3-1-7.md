# 解答

在 `FrequencyCounter` 中添加一个 `CountDistinct` 方法，计算不重复的键数。

```csharp
public static int CountDistinct<TKey>(TKey[] keys, IST<TKey, int> st)
{
    int distinct = 0;
    for (int i = 0; i < keys.Length; i++)
    {
        if (!st.Contains(keys[i]))
            st.Put(keys[i], distinct++);
    }
    return distinct;
}
```
