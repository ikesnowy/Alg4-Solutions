# 解答

书中已经给出了答案，当然在 Java 和 C# 里，`&&` 总是从左向右计算的，而且遇到 `false` 会直接返回结果。

如果数据结构中存在环，`IsOrdered` 有可能会陷入无限递归的情况，因此调用顺序比较重要。

# 代码

```csharp
public static bool IsBST(BST<TKey, TValue> bst)
{
    return IsBinaryTree(bst) &&
           IsOrdered(bst) &&
           HasNoDuplicates(bst);
}
```
