# 解答

上一题中的代码已经包含了对 Equals() 方法的实现。

# 代码

```csharp
/// <summary>
/// 比较两笔交易是否相同。
/// </summary>
/// <param name="obj">另一个对象。</param>
/// <returns></returns>
public override bool Equals(object obj)
{
    if (obj == this)
        return true;
    if (obj == null)
        return false;
    if (obj.GetType() != this.GetType())
        return false;
    Transaction that = (Transaction)obj;

    return
        (that.Amount == this.Amount) &&
        (that.When.Equals(this.When)) &&
        (that.Who == this.Who);
}
```
