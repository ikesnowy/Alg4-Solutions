# 题目

1.4.6
给出以下代码段的运行时间的增长数量级（作为 N 的函数）。
a.

```java
int sum = 0;
for (int n = N; n > 0; n /= 2)
    for (int i = 0; i < n; i++)
        sum++;
```

b.

```java
int sum = 0;
for (int i = 1; i < N; i *= 2)
    for (int j = 0; j < i; j++)
        sum++;
```

c.

```java
int sum = 0;
for (int i = 1; i < N; i *= 2)
    for (int j = 0; j < N; j++)
        sum++;
```

# 解答

a. N + N/2 + N/4 + … = ~2N，线性。

b. 1 + 2 + 4 + … = ~2N，线性。

c. NlogN，线性对数。
